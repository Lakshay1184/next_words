# ⚡ Next Word Predictor (LSTM)

A simple **Next Word Prediction web app** built using **LSTM (Deep Learning)** and **Streamlit**.
It provides real-time word suggestions similar to mobile keyboards.

---

## 🧠 Model

* Type: LSTM (Sequence Model)
* Framework: TensorFlow / Keras
* Trained on thousands of parameters
* Built using AI-based sequence learning
* **Accuracy: ~82%**
